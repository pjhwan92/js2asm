# js2asm
Javascript LLVM FrontEnd targeting Emscripten (personal project)

## Requirements
* Python 2.7
* Antlr4
* llvmpy 0.12.7

## Execution
$> python ECMAScriptMain.py [file name]

## Current progress
* Syntax pass
* Primitive type inference
* Base arithmetic operation
